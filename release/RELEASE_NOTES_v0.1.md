# ArenaForge Prototype v0.1

## Included in this build

- browser-playable auto-battler run
- shop, bench, board, merge, level, and economy systems
- creep, draft, loot, armory, elite, and boss rounds
- NPC opponent styles, encounters, augments, and run history
- web-ready main menu, settings, and credits

## Recommended play conditions

- play in landscape
- target resolution: `1280x720` or higher
- browser fullscreen is recommended

## Known limitations

- this is still a prototype build
- balance is not final
- some UI/art elements are still placeholder-level
- touch support is usable but not fully polished
- audio/presentation polish is still limited

## Focus of this release

- stable full singleplayer run
- browser demo for testing and feedback
- foundation for future TFT-like expansion
