# ArenaForge itch.io Release Notes

## Version

Prototype Web Release

## Included

- Full browser-playable autobattler run
- Shop, bench, board, XP, lock, traits, items, augments
- Draft, loot, armory, elite, and boss rounds
- NPC opponents with different styles and formations
- Run history and basic profile progress

## Known limitations

- Balance is still in progress
- Combat readability and VFX can improve further
- Touch support is partial
- PvP is not part of this release scope

## Recommended play mode

- Landscape
- 1280x720 and above
- Desktop browser preferred for the first public build
